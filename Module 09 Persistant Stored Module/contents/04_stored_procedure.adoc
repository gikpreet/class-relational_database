= 저장 프로시저

많은 데이터베이스 시스템은 데이터 처리에 필요한 문장들을 하나의 절차로 묶어서 처리할 수 있는 저장 프로시저를 제공합니다. 저장 프로시저를 사용하면 특정 데이터 처리 작업을 추상화하여 프로시저화 할 수 있으며, 처리 속도를 개선할 수 있습니다.
