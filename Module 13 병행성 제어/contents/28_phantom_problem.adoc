= 동적 데이터베이스와 팬텀 문제

image:./images/image05.png[]

* 트랜잭션 T1의 동작에 따라,
** 인덱스가 없어서 파일에 있는 모든 페이지를 모두 살펴보아야 한다면, T1은 기존 페이지들을 모두 잠금하고 파일에 새 페이지가 추가될 수 없도록 만들어야 함
** 인덱스가 있다면 등급이 1인 데이터 엔트리에 잠금을 할 수 있음. T2는 T1의 잠금을 해제를 기다려야 함. 이를 인덱스 잠금(Index Locking)이라고 함
