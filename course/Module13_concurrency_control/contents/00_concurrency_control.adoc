= Module 12: 병행성 제어

데이터베이스 관리 시스템은 조직내의 거의 모든 사용자가 동시에 사용하는 다중 사용자 시스템으로, 개별 사용자 프로그램이 동시에 조작하는 데이터에 대한 무결성을 위한 병행성을 제어할 수 있어야 합니다. 데이터베이스 관리 시스템에서는 트랜잭션을 사용하여 병행성을 제어합니다. 이 단원에서는 데이터베이스 관리 시스템의 병행 제어와 복구를 알아봅니다.
