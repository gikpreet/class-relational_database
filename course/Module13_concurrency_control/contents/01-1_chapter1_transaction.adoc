= Chapter 1: 트랜잭션

트랜잭션(Transaction)은 데이터베이스에서 하나의 작업 단위를 의미하며, 논리적으로 묶인 일련의 연산들을 포함합니다. 트랜잭션은 데이터 무결성을 유지하고, 안전한 데이터 처리를 보장하는 중요한 개념입니다. 여기에서는 트랜잭션의 개요에 대해 알아봅니다.
