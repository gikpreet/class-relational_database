= ER 모델 기능

ER 모델은 데이터의 중요한 특징들을 표현할 수 있는 요소들을 가지고 있습니다. 그 기능들을 사용하면 모델링하려는 조직체의 여러 가지 의미를 관계 모델보다 더 잘 설명할 수 있고, 이는 관계형 데이터베이스 설계에 ER 모델을 사용하는 주된 이유이기도 합니다. 이 단원에서는 ER 모델의 기능들에 대해 살펴봅니다
