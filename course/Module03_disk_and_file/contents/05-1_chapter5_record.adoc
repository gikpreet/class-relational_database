= Chapter 5: 레코드 형식

주 기억장치의 예약된 공간인 버퍼 풀은 프레임이라는 페이지가 로드될 수 있는 공간을 제공하고, 버퍼 관리자는 디스크 관리자로부터 전달받은 페이지를 버퍼 풀에 로드합니다. 페이지 내의 데이터는 레코드라는 형식으로 구성됩니다. 이 장에서는 레코드의 형식에 대해 공부합니다.
