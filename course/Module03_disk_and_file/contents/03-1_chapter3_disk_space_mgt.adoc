= Chapter 3: 디스크 공간 관리

DBMS 소프트웨어 구성요소의 가장 아래에는 디스크 관리자가 존재합니다. 디스크 관리자는 데이터의 단위로 페이지라는 개념을 지원하며, 페이지를 할당하고 반환하는 명령과 페이지를 판독하고 기록하는 기능을 제공합니다. 이 장에서는 디스크 공간 관리에 대해 학습합니다.
